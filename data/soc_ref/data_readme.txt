/ground_truth contains data files with human coding of look/no_looks for subset
of frames

results_openFace_8_17.csv contains the features that were output from OpenFace using
an older version of the model (based on docker image on their site)

results_openFace_03_3_18.csv contains the features that were output from OpenFace using
the newest version of the model (installed locally)

socref_sample_phases.csv contains information about the timing of trials and
looks in the social referencing experiment. these are used to see how the model
is doing at coding looks during key points of the experiment.
